SynxDB 2 Documentation

    

gpv

Configures, deploys, and manages SynxDB clusters on top of a virtual platform such as VMware vSphere.

Synopsis

gpv base <base_options>
gpv config <config_options>
gpv greenplum <greenplum_options>
gpv version
gpv -h | --help

Description

The gpv utility automates the deployment of a SynxDB cluster on top of virtual platforms such as vSphere. You use gpv to specify a SynxDB cluster’s configuration; automate the deployment of a base virtual machine, clone the base virtual machine to create a cluster, and initialize a SynxDB on the set of virtual machines.

Options

gpv base The base virtual machine commands.

gpv config The commands to manage the configuration of the SynxDB deployment.

gpv greenplum The SynxDB initialization and validation commands.

version Print the version information for the gpv utility.

-h, –help

Display the online help.
